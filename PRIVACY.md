# 隐私说明

同频 Clean 用于把手机当前媒体状态同步到用户自行部署的服务器，并通过房间码和房间密钥读取状态或发送播放控制命令。

## 可能处理的数据

- 当前歌曲名称、歌手、专辑、播放进度、播放状态、播放器包名和播放器名称。
- 用户主动开启歌词增强后读取到的当前歌词与下一句歌词。
- 用户主动填写的歌曲分享链接。
- 房间内主动添加的听歌笔记。

## 不会主动上传的数据

- QQ 音乐、酷狗音乐、网易云音乐或其他应用的屏幕截图。
- 通讯录、短信、相册、麦克风或精确位置。
- 其他应用中的聊天内容。

本地 OCR 截图仅在设备内存中处理，不保存、不上传。QQ 音乐歌词读取和 OCR 默认关闭，需要用户在 App 与 Android 系统设置中主动开启；酷狗音乐和网易云音乐当前不读取界面内容，只通过系统媒体会话同步基础播放状态。

## 数据保存位置

每位测试者应自行部署服务器。默认免费 Render 配置把房间数据保存在临时文件系统中，服务重启或重新部署后数据可能消失。房间密钥等同于房间密码，请勿公开分享。

## 第三方服务

- Render：托管用户自行部署的服务器。
- LRCLIB：按歌曲信息查询公开歌词。
- GitHub Actions：构建测试版 APK。

使用者应同时阅读并遵守相应第三方服务的条款与隐私政策。
