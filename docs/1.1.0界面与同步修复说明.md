# 同频 Clean 1.1.0 更新说明

## UI 重构

- 首页改为播放器主卡片：封面、歌名、歌手、专辑、进度、当前歌词和下一句集中展示。
- 新增本机上一首、播放/暂停、下一首控制。
- 房间码突出显示，房间密钥默认隐藏。
- 服务器、权限、分享链接和诊断信息收进独立卡片。
- 新增四套主题：奶油白、星空蓝、抹茶绿、雾紫。
- 新增应用图标和更清晰的同步状态提示。

## 歌词修复

- 切歌时立即取消上一首歌词任务，避免旧请求堵住新歌。
- 多个歌词匹配请求并行执行，总等待时间限制在约 5 秒。
- 自动清洗 `cover:`、翻唱、Live、现场版等标题标记。
- 支持从 `cover:原唱` 中提取原唱歌手，提高翻唱歌曲匹配率。
- 匹配失败后显示明确结果，不再无限停在“歌词加载中”。
- 新增“重试歌词”按钮；失败结果短暂缓存后允许再次匹配。

## 控制与同步修复

- 远程命令不再以“按键已发送”冒充成功。
- 手机会检查播放状态、歌曲或进度是否真的变化，再回报 executed。
- 播放器响应慢时自动使用媒体按键重试一次。
- 最长等待约 10 秒，仍无变化时回报 failed，并给出明确提示。
- 轮询缩短到约 700ms；切歌和控制后会连续抓取状态。
- “刷新状态”增加处理中、成功和超时反馈。
