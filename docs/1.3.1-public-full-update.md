# 1.3.1 Public Full 更新说明

本版将原私用版中适合公开发布的通用功能合并回公开版，同时保留 1.3.0 的多播放器 OCR 与局域网部署能力。

## 合入功能

- 手机端底部导航：播放 / 笔记 / 设置。
- 独立听歌笔记页：支持当前歌曲过滤、全部笔记、点击当前歌曲笔记跳转进度、长按复制。
- 手动歌词导入：支持粘贴或选择 `.lrc / .txt` 文件，按当前歌曲保存到本机。
- 歌词优先级：手动导入 > LRCLIB > 播放器界面 > OCR > 未找到。
- MCP 新增 `search_and_play`。
- 网页遥控器新增搜索并自动播放入口与最近听歌笔记列表。
- 服务端继续保留 `/lan` 自检与局域网部署说明。

## 播放器范围

- QQ 音乐、酷狗音乐、网易云音乐：状态读取、基础媒体控制、进度同步、界面歌词读取、OCR 兜底。
- QQ 音乐：自动点歌无障碍兜底。
- 酷狗 / 网易云：自动点歌先尝试系统媒体搜索；后续可继续适配 UI 自动搜索。

## 隐私说明

- 手动歌词只保存在手机本机。
- OCR 截图只在本机内存中临时处理，不上传、不保存。
- 房间密钥仍等同房间密码，不应公开。
