# 1.1.3 多 AI 连接说明

本次更新不改变核心播放同步链路，重点是把公开测试版从“像是只给 ChatGPT 用”调整为“任何支持 MCP、REST API 或网页遥控器的 AI / 工具都能参与”。

## 新增

- 新增 `docs/AI连接教程.md`。
- README 增加“多 AI 连接方式”速览。
- 网页遥控器文案改为“不用接入任何 AI，也能先测试手机控制链路”。
- Android 房间提示改为“AI 客户端或网页遥控器”，不再只写 ChatGPT。
- MCP 工具说明补充适用范围。
- 版本升级至 1.1.3，Actions 产物名为 `tongpin-clean-1.1.3-apk`。

## 连接方式

1. 远程 MCP：适合支持自定义 MCP / connector 的 AI 客户端。
2. REST API：适合 Dify、Coze/扣子、n8n、自建 Bot 或其他能发 HTTP 请求的平台。
3. 网页遥控器：适合不支持工具调用的普通聊天 AI 用户，也适合先排查手机控制链路。

## 兼容性

- 服务端 REST API 与 1.1.2 兼容。
- 手机 App 核心同步逻辑与 1.1.2 一致。
- 歌词增强仍主要针对 QQ 音乐。
- 本次没有加入酷狗 / 网易云适配；多播放器适配建议另起 1.2.x 或 1.3.x 分支处理。
