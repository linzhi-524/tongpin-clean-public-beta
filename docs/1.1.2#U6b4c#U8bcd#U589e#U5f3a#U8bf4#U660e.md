# 1.1.2 歌词增强说明

## 新增

- QQ 音乐专用无障碍歌词读取服务；同时尝试读取歌词页和系统可见的 QQ 音乐悬浮歌词节点。
- 本地 ML Kit 中文 OCR 兜底（Android 11 及以上）。
- App 设置页增加歌词读取与 OCR 独立开关。
- 诊断页显示歌词读取授权与 OCR 状态。
- 当前歌词优先级：新鲜的 QQ 音乐界面/OCR 结果 > LRCLIB 时间轴结果。

## 隐私边界

- 功能默认关闭。
- 无障碍服务只声明监听 `com.tencent.qqmusic`。
- OCR 截图仅在本机内存中裁剪与识别，识别完成后立即释放。
- 不上传、不保存截图；房间仅同步当前句、下一句、来源与时间。

## 兼容性

- Android 8+ 可使用界面文字节点读取。
- Android 11+ 可使用无障碍截图与 OCR 兜底。
- QQ 音乐若不暴露歌词文字节点，会自动尝试 OCR（需用户开启）。
