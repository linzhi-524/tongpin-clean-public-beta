# 同频 Clean 1.3.0 Public 更新说明

本版重点是把公开版继续保持干净通用，同时补上多播放器歌词兜底和局域网部署能力。

## 新增

- QQ 音乐、酷狗音乐、网易云音乐都可尝试界面歌词读取。
- 三款播放器都可在文字节点失败后启用本地 ML Kit 中文 OCR 兜底。
- App 支持局域网 HTTP 服务端地址，例如 `http://192.168.1.100:3000`。
- 设置页新增“局域网部署”和“填入局域网示例”入口。
- 服务端新增 `/lan` 自检说明接口。
- 新增 `docs/LAN_DEPLOY.md` 局域网部署教程。

## 保留

- Render 一键部署。
- GitHub Actions 构建 Debug APK。
- MCP / REST API / 网页遥控器三种连接方式。
- LRCLIB 时间轴歌词匹配。
- 房间码、房间密钥、听歌笔记和远程播放控制。

## 隐私边界

- 歌词读取和 OCR 默认关闭。
- OCR 截图只在本机内存中裁剪识别，识别后立即释放。
- 服务端只接收当前句、下一句、歌词来源和识别时间，不上传屏幕截图。

## 兼容性说明

- 酷狗音乐和网易云音乐的界面结构可能随版本变化，OCR/文字节点读取需要真机验证。
- 局域网 HTTP 仅建议在可信 Wi-Fi 内使用；远程使用建议部署到 Render HTTPS 地址。
